# Data Source: apono_spaces

Retrieves existing Apono Spaces. Use this data source to reference spaces in the Access Flow, Bundle, or Access Scope resources.

## Example Usage

### Retrieve All Spaces

```terraform
data "apono_spaces" "all" {
}
```

### Retrieve a Specific Space by Exact Name

```terraform
data "apono_spaces" "production" {
  name = "Production"
}
```

### Retrieve Spaces Matching a Pattern

```terraform
data "apono_spaces" "engineering_spaces" {
  name = "Engineering*"
}
```
