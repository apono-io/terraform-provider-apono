# Data Source: apono_space_scopes

Retrieves existing Apono Space Scopes. Use this data source to reference existing space scopes when creating or updating spaces.

## Example Usage

### Retrieve All Space Scopes

```terraform
data "apono_space_scopes" "all" {
}
```

### Retrieve a Specific Space Scope by Exact Name

```terraform
data "apono_space_scopes" "production_aws" {
  name = "Production AWS"
}
```

### Retrieve Space Scopes Matching a Pattern

```terraform
data "apono_space_scopes" "aws_scopes" {
  name = "*AWS*"
}
```
