# Resource: apono_space_scope

Manages an Apono Space Scope, an AQL expression that defines which resources are visible within a space.

## Example Usage

### Scope by Integration Type and Tag

```terraform
resource "apono_space_scope" "production_aws" {
  name  = "Production AWS"
  query = "integration in (\"aws-account\") and resource_tag[\"environment\"] = \"production\""
}
```

### Scope by Resource Name Pattern

```terraform
resource "apono_space_scope" "staging_databases" {
  name  = "Staging Databases"
  query = "resource_type = \"aws-rds-mysql-database\" and resource_name contains \"staging\""
}
```

### Scope Combining Multiple Integration Types

```terraform
resource "apono_space_scope" "cloud_production" {
  name  = "Cloud Production"
  query = <<EOT
  (integration in ("aws-account") or integration in ("gcp-project"))
    and resource_tag["environment"] = "production"
  EOT
}
```
